# Feature Matching OpenCV2

Feature detection and matching using OpenCV 2.4

## Library Dependencies   

- OpenCV C++ version 2.4.x
- CMake


## Getting Started  

**Compiling**  

The following libraries are required to compile the code.    

1. `$ mkdir build`
2. `$ cd build`
3. `$ cmake ..`
4. `$ make`  

If the code is compiled successfully, you should see a executable in the folder `bin/linux` (Linux) or `bin/macos` (Mac):  

`FeatureMatching`

**Run feature matching**  

Use the following command line to run feature matching with SIFT.

`$ ./FeatureMatching`
